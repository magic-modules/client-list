@magic-modules/library-list

this is the @magic-modules LibraryList component. shows a list of all @magic-modules with gitbadges and urls to the docs

installation

npm install magic-modules/library-list

require

first add the component to the assets

// assets/index.jsmodule.exports = {  // ..other exports  LibraryList: require('@magic-modules/library-list'),}

usage

in a page or component

module.exports = () => [LibraryList]

@magic-libraries libraries

below is a collection of the available @magic client libraries.

  • @magic-libraries/json

    the @magic-libraries/json module parses and stringifies json. it also returns errors instead of throwing them.

    docs / demo

  • @magic-libraries/is

    the @magic-libraries/is module unifies the javascript type testing apis.

    docs / demo

  • @magic-libraries/uri

    the @magic-libraries/uri module encodes objects to uri strings and decodes uri strings to objects.

    docs / demo
